Enum commands to define register addresses and register most-\/used values. 

These enum commands either point to a hexadecimal register address for the M\+P\+U-\/6050 or hexadecimal values most used to write towards a register. If you want to read or write to a register, it is highly recommended to use the enum commands to avoid making mistakes. Accelerometer and gyroscope registers consist of 2 bytes. The \+\_\+H is the M\+SB and the \+\_\+L is the L\+SB.
