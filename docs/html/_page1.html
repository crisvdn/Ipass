The following naming conventions are used:

Function

This page contains the subsections read functions. and Display functions..

read functions.

All functions that start with read, return the value insided a user defined variable. It is used so the user can decide what to do with this information. For example: show on any kind of display or manipulate data with his/her own functions.
Examples:

read_accelX();
read_accelY();
read_accelZ();
read_bytes();
read_pitch();
read_roll();
read_values();
read_word();

Display functions.

All functions that start with display, outputs the data towards the screen using hwlib::cout.
Examples:

display_calibrate_values();
display_raw_values();
display_roll_pitch();
display_values();

Set and Get functions.

All functions that start with set and get, either returns the value towards the datatype mentioned, or sets the value the user has inserted.
Examples:

set_accel_gyro_scale();
set_calibrate_values();
set_register();
